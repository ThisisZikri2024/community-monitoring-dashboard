🛡️ Titan EMS v12.0 - Ultimate Oversight Edition
Titan EMS (Enhanced Management System) adalah sistem pengurusan ahli dan pemantauan keselamatan digital berasaskan web yang ringan namun berkuasa. Ia direka khas untuk pentadbir yang memerlukan kawalan mutlak terhadap disiplin pengguna, terutamanya dalam memantau aktiviti media sosial bagi kategori bawah umur.

🚀 Ciri-Ciri Utama
Manual Oversight Engine: Pentadbir boleh memasukkan sebab kesalahan atau nota pemantauan secara manual tanpa had pilihan dropdown yang kaku.

Protection Zone (Underage): Mod khas untuk pengguna bawah umur yang membolehkan sekatan aplikasi media sosial (Instagram, TikTok, Roblox, dll.) secara individu.

Criminal Archive (Audit Logs): Sejarah pemadaman profil tidak akan hilang; ia akan disimpan dalam arkib jenayah untuk rujukan audit keselamatan.

One-Click CSV Export: Jana laporan lengkap dalam format .csv yang sedia untuk dibuka dalam Microsoft Excel atau Google Sheets.

Modern Responsive UI: Antaramuka gelap (Sidebar) yang kemas dengan statistik "real-time" menggunakan CSS Grid dan Flexbox.

Zero-Server Architecture: Menggunakan LocalStorage sepenuhnya—data anda kekal dalam pelayar tanpa memerlukan database atau hosting yang mahal.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Variables, Plus Jakarta Sans Font)

Logic: Vanilla JavaScript (ES6+)

Storage: Browser LocalStorage API

Export: Blob & URL Object API (CSV Engine)
